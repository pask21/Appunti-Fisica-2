# Appunti di Fisica 2

Raccolta personale di appunti del corso di **Fisica 2**, scritti in LaTeX e organizzati secondo il consueto ordine accademico. 
L'obiettivo di questa repository è mantenere materiale ordinato, chiaro e facilmente riutilizzabile per studio e ripasso.

## Contenuto

Gli appunti coprono principalmente:

- Elettrostatica
- Magnetostatica
- Equazioni di Maxwell
- Elettrodinamica
- Onde elettromagnetiche
- Energia e vettore di Poynting
- Esercizi svolti

## Struttura della repository

```
.
├── build/
    ├── figure/
    └── appunti_fisica2.pdf
├── appunti_fisica2.tex
└── preambolo.tex
```

- `figure/` contiene immagini e diagrammi TikZ
- `appunti_fisica2.pdf` pdf già compilato con il tex corrente
- `appunti_fisica2.tex` file principale per la compilazione
- `preambolo.tex` contiene pacchetti e comandi personalizzati

## Obiettivo

Questi appunti sono pensati per:

- Ripasso rapido
- Preparazione esami
- Consultazione formule
- Raccolta dimostrazioni importanti

## Note

Gli appunti sono in continuo aggiornamento e potrebbero contenere errori o parti incomplete.
